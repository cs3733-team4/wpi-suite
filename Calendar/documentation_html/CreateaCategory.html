Create a Category


To create a new Category:


1. Click on Manage Categories button near the top of the module.



2. Enter a desired name and color for your new Category. Both fields are required for creating a new Category.



3. Click Save.


The new Category will be listed on the list on the left panel, displaying the chosen Category name and color.
