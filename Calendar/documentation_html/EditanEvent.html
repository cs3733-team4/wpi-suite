Edit an Event


To edit an Event from any view ( Day, Week, or Month):


1. Select an Event. Its details should appear in the Details Pane on the bottom left.



2. Below its details within the Details Pane, select Edit. Alternatively, double-click the Event in the calendar view.

3. Edit the information in the desired field.

4. Select Save.

The Event has now been changed, and selecting it from within any view will display the updated information.
