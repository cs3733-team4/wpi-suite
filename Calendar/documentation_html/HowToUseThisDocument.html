How to Use This Document


There are several formatting standards used throughout this documentation set. Note that when viewed inside Team 4's Calendar Application, this help document is interactive with the application. Formats marked with a "*" are only applicable while inside the application.


Personal Action Required:

Anytime this format is used, you expected to perform the action.  For example in the sentence: "Click the 'New Event' button to make a new a tab", you are expected to click on the 'New Event' Button.


Concept Reference:

       When you see a keyword with this formatting, you can click the text to navigate to a description of the concept.  For example in the sentence: "The Event is now saved on the database, and will appear on the calendar", clicking on 'Event' will bring you to a page describing what an Event is.


Navigation Link:

Links formatted this way are used for navigating this document. They are only used to jump between help pages.


Clickable Action Item*:

Text appearing like this has the ability to do the action that it implies.  For example in the sentence: "Click the 'New Event' button to make a new a tab", clicking on the 'New Event' will make the new tab for you.


External Web Link:

       These links navigate to external web-pages and are no longer inside the documentation manager.
