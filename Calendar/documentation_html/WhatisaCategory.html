What is a Category?


A category is a way of conceptually organizing events and to allow a user to filter Events very quickly. A Category has only a name and an associated color; when an Event is selected, the color representing its Category is displayed in the Details Pane behind the Event title, as well as on the calendar for each event. For example, a Category can be created for holidays, meetings, or work.
