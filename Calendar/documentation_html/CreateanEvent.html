Create an Event


To create an Event:


1. Click on Add Event in the ribbon bar.


2. The Name, Start Date, Start Time , End Date, and End Time fields are the minimum requirements for creating an event. Participants, Category, and Description are optional.


a. In order to save the Event, appropriately fill at least the required fields. For example:



b. Formatting for Date and Time are as follows:

1. Date: MM/DD/YYYY

2. Time: HH:MM (Based on 12-hour clock, permuted with AM/PM)


3. By default, the Category field defaults to 'Uncategorized' and Project Event is checked. To assign an event to a category, click the drop down menu and select a category. To create an event on the Personal Calendar, select the 'Project Event' check box.

4. Once all the data is entered, click on Save to save you new event.


The Event should now appear on the calendar on the specified time and day for the specified length of time.
