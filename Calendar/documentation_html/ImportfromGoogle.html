IMPORT FROM GOOGLE CALENDAR

To import events from Google Calendar:
    1. Select the Import from Google button in the top right.



    2. Enter your Google username and password into the username and password fields and submit. Be careful to type your username and password correctly, or the program will not be able to log you in.



Your events have now been imported from Google Calendar. Each Event is now individually editable. WARNING: Any changes to Events in WPI Suite will not sync back to your Google calendar. This is a one-way import only. Importing from the same Google Calendar will not create duplicates of each Event.
