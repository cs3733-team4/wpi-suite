What is an Event?


An Event is a scheduled slot of time dedicated to an occurrence. An Event contains:


• Title*

• Start Date & Time*

• End Date & Time*

• Participants

• Category

• Description


An event can be saved on a personal calendar or a team's project calendar. For example, an event can be created on a calendar to book a time for a meeting on the team project calendar, and another event can be created to mark a time for a dentist appointment on a personal calendar.


(* indicates a required entry)
