Edit a Category


To edit an existing Category:


1. Click on Manage Categories button near the top of the module.



2. Select a field desired field and edit either name, color, or both for the selected Category. Both fields are required to save changes to the Category.



3. Click Save.


The edited Category will be listed on the list on the left panel displaying the chosen Category name and color.
