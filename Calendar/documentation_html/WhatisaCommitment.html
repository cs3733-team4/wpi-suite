What is a Commitment?


A Commitment is a representation of deadlines. A Commitment contains:


• Title*

• Due Date*

• Due Time*

• Participants

• Description


Commitments can be saved only on a personal calendar (Team YOCO will eventually implement team commitments). For example, a Commitment can be created to represent a deadline for a report.


(* indicates a required entry)
