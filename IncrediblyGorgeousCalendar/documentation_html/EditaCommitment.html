Edit a Commitment


To edit a Commitment from any view ( Day, Week, or Month):


1. Select a Commitment. Its details should appear in the Details Pane on the bottom left.

2. Below its details within the Details Pane, select Edit. Alternatively, double-click the Commitment. A tab will appear with the information of the selected Commitment.



3. Edit the information in the desired field.

4. Select Save.

The Commitment has now been changed, and selecting it from within any view will display the updated information in the Details Pane.
