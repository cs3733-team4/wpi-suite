Create a Commitment

To create a new Commitment:

1. Click on Add Commitment in the ribbon bar.



2. The Name, Date, and Time fields are the minimum requirements for creating a Commitment. Participants and Description are optional fields.


a. In order to save the Commitment, appropriately fill at least the required fields.

b. Formatting for Date and Time are as follows:


1. Date: MM/DD/YYYY

2. Time: HH:MM (Based on 12-hour clock, permuted with AM/PM)


3. Select Save.



The Commitment should now appear on the calendar on the specified day and at the specified time.
